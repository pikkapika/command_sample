# awkのコマンドリファレンス

## 出力
- print: フォーマット無し。改行は自由にできる。
- printf: フォーマットを指定できる。%-幅.桁数f

## 正規表現
- ^
- $
- .     改行をのぞく任意の１文字
- [..]  文字セットスクエアブラケットで囲まれた文字の任意の 1 文字
- [^..] スクエアブラケットで囲まれた文字以外の任意の 1 文字
- |     選択演算子，選択を指定
- (..)  正規表現のグループ化
- *     直前の正規表現の 0 回以上の繰り返し
- +     直前の正規表現の 1 回以上の繰り返し
- ?     直前の正規表現の 0 回または 1 回のマッチ

## 変数

- $0: レコード全体
- $n: nフィールド目
- NF: 現在の入力レコードのフィールド数
- NR: プログラムが始まってから今までに読んだレコード数
- FS: 入力フィールドセパレータ．デフォルトはホワイトスペースのシーケンス
- RS: レコードセパレータ．デフォルトはニューライン

## 制御文
- if
    if (condition) {} else {}
- while
    while (condition) {}
- for
    for (initialization; condition; increment) {}
    for (var in array ) {}

## 演算子
- x ~ y     文字列 x が正規表現 y とマッチする場合，真
- x !~ y    文字列 x が正規表現 y とマッチしない場合，真

## 組み込み関数(文字列操作)
- index(in, find)
- length(string)
- sub(regexp, replacement, target) target の中で，正規表現 regexp にマッチする一番左の部分文字列を探し，それを replacement で置換して，文字列全体を変更します
- gsub(regexp, replacement, target) sub() の強力版で，部分文字列を全て置換します
